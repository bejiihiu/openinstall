OpenInstall v0.1.0

A Linux app installer that works across distros. One CLI, one GUI — apt, dnf, pacman, zypper, all the same way.

⬇ Install OpenInstall

Install

curl -sSf https://raw.githubusercontent.com/bejiihiu/openinstall/main/scripts/install.sh | sh

Detects your architecture (x86_64 / aarch64), downloads the binary, and registers desktop entry + URI handlers automatically.

Quick start

# Detect your system
installer detect

# Install from a local manifest
installer install ./cursor.json

# Install directly from a URL
installer openinstall://cursor?m=https://example.com/cursor.json

CLI reference

installer detectPrint distro, arch, package manager
installer validate <manifest>Check manifest format
installer select <manifest>Show matching package
installer show <manifest>Print app details
installer verify <manifest>Check sha256 + signature
installer install <manifest>Download, verify, install
installer remove <manifest>Uninstall
installer update <manifest>Update (same as install)
installer reinstall <manifest>Force reinstall
installer rollback <manifest>Revert to cached version
installer history <manifest>Installation history
installer cache clearNuke the cache
installer cache infoShow cache size
installer publish --name ...Generate a manifest
installer serve <manifest> [addr]Serve /app/latest on HTTP
installer gui [manifest]Launch graphical installer
installer gui --register-desktopRegister in app menu + URI handlers
installer self-updateDownload and replace itself

How it works

OpenInstall detects your distro, downloads the right package, verifies it, then hands it to your system package manager. Three crates, one dependency chain:

installer-core          types, adapters, verification, runtime
installer-cli           CLI (20+ commands) + optional GUI
installer-bootstrapper  Tiny entry point for download + launch

See architecture docs for details.
